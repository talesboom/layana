# README
## Installation 
- Ruby version 2.2.0+ 
- Rubygems 
- Bundler
- Rails 5+
- git

## Database 
- postgres

## Getting Start
get start with the following steps
### Set up git
- ```$ git config --global user.name "Your Name"```
- ```$ git config --global user.email your.email@example.com```
### Clone git
- ```$ git clone https://github.com/talesboom/layana.git```
